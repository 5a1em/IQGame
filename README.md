# IQGame

### Управление
1. Зажимает ЛКМ на шарике, который мы хотим переместить
2. Выбираем направление, не отпуская ЛКМ (стрелочка на мячике указывает куда полетит мячик) 
3. Отпускаем ЛКМ, в результате чего, мячик перемещается
